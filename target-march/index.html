三月总目标

考研复习

  1. 微积分第一轮复习结束，线代第一轮复习进行一半；
  2. 英语阅读每日一篇精读，争取将错误控制在2个以内；
  3. 英语单词每日半个unit和2组，记录每日阅读的生词句子；
  4. 英语每日复习前一日单词及阅读；

项目

  1. 完成需求分析；
  2. 确定需要学习技术，基本了解技术知识；
  3. 完成前端界面；
  4. 设计核心算法；

技术学习

  1. JS一周内了解基本编程知识，学习node.js框架；
  2. 熟练掌握markdown；
  3. 自定义blog主题，熟悉相关配置；
  4. 掌握latex文档编写；
  5. 坚持每周至少更新2篇博客；

运动

  1. 坚持每天慢跑及运动；
  2. 坚持每周六完成一次10公里以上的步行；
